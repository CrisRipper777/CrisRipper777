实验题目：利用云平台搭建个人博客
实验学时：2 实验日期：2023.3.14
实验目的：熟悉个人博客系统的搭建。 具体包括： 参考方案：注册Github账号，搭建Hexo环境并实现个人博客搭建，撰写实验报告。
硬件环境： 联网的计算机一台
软件环境： Windows or Linux Node.js Hexo
实验步骤与内容： 实现个人博客系统的搭建： 1. 首先配置环境，采用的是参考方案，使用Github以及Hexo环境实现个人博客搭建。这里需要配置Node.js和Git的环境，由于之前配置过，这里可以直接配置Hexo环境，Hexo是一个基于 node.js的快速生成静态博客的开源框架,支持 Markdown和大多数 Octopress插件,一个命令即可部署到 Github页面、 Giteee、 Heroku等,强大的APl,可无限扩展,拥有数百个主题和插件。 使用 npm install -g hexo-cli 来安装hexo： 2. 之后使用如下命令： 首先hexo init 是初始化hexo环境,这时会在目录下自动生成hexo的文件，然后的Hexo generate命令会生成静态页面文件，最后hexo server 是在本地部署我们的页面，按照提示浏览器打开http://localhost:4000 即可看到示例页面： 3. 之后只需要参考hexo的官方文档，对搭建的博客页面进行修改与配置等，即可成功搭建起自己的博客主页。 4. 最后将本地博客发布到网络上，这里使用的是GitHub，首先在GitHub上创建一个仓库命名为：CrisRipper777(用户名).github.io，之后在本地的文件夹中打开_config.yam文件，在文件中配置我们的如下部分： 之后在Git Bash中使用命令npm install hexo-deployer-git –save，以及hexo deploy 即可将我们的博客网页的文件保存到我们所指定的Github仓库中： 最后打开网址https://crisripper777.github.io/ 即可得到一个和上图一样的博客网页。
结论分析与体会： Hexo是一个基于 node.js的快速生成静态博客的开源框架,支持 Markdown和大多数 Octopress插件,一个命令即可部署到 Github页面、 Giteee、 Heroku等,强大的APl,可无限扩展,拥有数百个主题和插件。 GitHub Pages，是用来托管 GitHub 上静态网页的免费站点
